# Spec 064-multi-type-within

## Titulo
Multi `type ... within` container resolution

## Motivacion
En SR* con varios `type X within Y` la atribucion del contenedor debe usar anidacion real.

## Alcance
- Cambios localizados en server.
- Sin nuevas features de usuario.

## Criterios
- Tests verdes.
- Cero regresiones.
